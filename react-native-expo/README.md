A beautifully designed, gamified Hydration and Solfeggio Audio Meditation companion ported to **React Native & Expo**. 

This folder contains the complete high-fidelity implementation corresponding exactly to your Android Jetpack Compose app!

1. **Self-Contained Navigation Drawer**: Frosted glassy navigation tab matching style aesthetics with gorgeous smooth icons.

## 🛠️ Step-by-Step Installation instructions

To launch and run this application inside **Expo Go** on your device, execute the following commands in your computer's terminal:

### 1. Extract the folder
Download this workspace as a ZIP file (or push the code to Github), then navigate into this folder on your system:
```bash
cd react-native-expo
```

### 2. Install Project Dependencies
Run npm (or yarn) to fetch exact package dependencies needed for layout grids, graphics rendering, and dynamic gradients:
```bash
npm install
```

This installs Expo Go compatible versions of:
* `react-native-svg` for beautiful vector fluid liquid waves sloshing.
* `lucide-react-native` for high-fidelity icons across all screens.
* `expo-linear-gradient` for color themes and visual glassy accents.

---

## 📱 How to Run the App

Start the Expo bundler:
```bash
npm run start
```

* **For iOS/Android Mobile Devices**: Download the **Expo Go** application from the App Store/Google Play. Scan the generated QR Code on your Terminal screen to see the fully dynamic app in action instantly!
* **For Web Viewers**: Press `w` on your dashboard console to launch a browser preview panel.
